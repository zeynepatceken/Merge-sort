# Merge-sort
Dizinin sort türüne göre aşamalarının yazımını ve Big-0 gösterimini yapınız.

[16,21,11,8,12,22] -> Merge Sort

Yukarıdaki dizinin sort türüne göre aşamalarını yazınız.
Big-O gösterimini yazınız.

[16,21,11] [8,12,22]

[16,21] [11] [8,12,22]

[16] [21] [11] [8,12] [22]


[8] [12]


[16,21] [8,11] [12,22]

[8,11,12,16,21,22]

Big-O gösterimi:
O(nlogn)
